# Student Marks Analyzer (Flask)

A simple Flask web application to calculate average marks of students.

## Features
- Input marks using a form
- Calculates average score
- Displays result on a separate page
- Handles invalid input gracefully

## Tech Stack
- Python
- Flask
- HTML

## How to Run
1. Clone the repo
2. Install Flask
3. Run `python app.py`
4. Open `http://127.0.0.1:5000/form`
